# Language Translator Chat

A lightweight, client-side language translator with chat-like UI, powered by JavaScript and the MyMemory API. Features include translation history and favorites saved locally via localStorage.

## Features
- ✅ Real-time translation using **MyMemory API**
- 📝 Translation history stored in browser localStorage
- ⭐ Favorite translations with one-click save & recall
- 📱 Mobile-responsive design
- 🧩 Pure JavaScript — no backend required

## Setup
1. Clone or download the repo
2. Open `index.html` in any modern browser
3. Start translating!

> ⚠️ Note: You may need to enable CORS or use a proxy if MyMemory API blocks direct browser requests.
